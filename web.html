Web design





HyperText Markup Language (HTML) is the set of markup symbols or codes inserted into a file intended for display on the Internet. The markup tells web browsers how to display a web page's words and images.



Cascading Style Sheets, fondly referred to as CSS, is a simple design language intended to simplify the process of making web pages presentable. CSS handles the look and feel part of a web page. This is easy to learn.



JavaScript is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive.JavaScript gives web pages interactive elements that engage a user.
